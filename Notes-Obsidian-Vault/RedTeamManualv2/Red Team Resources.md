# NMap
#pingsweep 
#disablearpping
Disables ARP discovery:
```bash
nmap -sn 0.0.0.0 --disable-arp-ping
```
Ping Sweep:
```bash
nmap -sP 192.168.1.0/24
nmap -sn 192.168.1.0/24 --packet-trace
```
Host Discovery:
```bash
nmap -sn 192.168.1.108 --packet-trace --send-ip
```
